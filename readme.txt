== Description ==

Welcome to the **Custom Blog Grid** plugin! This tool allows you to easily display your latest posts using a modern layout where one featured post stands out alongside four secondary posts. 

### How to Use (Instructions)

You can place the grid on any page, post, or widget area using our simple shortcode. 

**1. Standard Layout (Featured post on the left)**
Simply specify the category slug you want to display:
`[blog_grid category="aging-parents"]`

**2. Alternate Layout (Featured post on the right)**
Use the positioning parameter to flip the design:
`[blog_grid category="aging-parents" featured_position="right"]`

**3. Show Latest Posts from All Categories**
If you leave the category parameter empty, it will display the 5 most recent posts across your entire site:
`[blog_grid]`

### Available Parameters

* **`category`**: The exact slug of the category (e.g., `health`, `news`).
* **`featured_position`**: Accepts `left` (default) or `right`.
